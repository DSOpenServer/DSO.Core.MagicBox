Genel Bakış

MagicBox’ın temel fikri: değerleri doğrudan saklamak yerine değer üreten (lazy) Func<T> hücreleri saklamak. Böylece, tip güvenliği korunurken “hot-path” erişimlerde DynamicInvoke maliyetinden kaçınılır.

  • Typed access: Get<T>, TryGet<T>
  • Generic olmayan erişim: GetVal, GetString (cache ile hızlı yol)
  • Tablo soyutlaması: DDContainer içinde tablolara/satırlara ayrılmış model

Kurulum

Repo “core” kütüphane mantığında: solution’a proje referansı veya kaynak ekleme ile kullanılabilir. 

// Proje referansı (önerilen)
<ItemGroup>
  <ProjectReference Include="..\DSO.Core.MagicBox\DSO.Core.MagicBox.csproj" />
</ItemGroup>

Hızlı Başlangıç

Hücre ekleme, tip-güvenli okuma ve “string/object” hızlı yolu: 

using DSO.Core.MagicBox;

var box = new DMagicBox();

// Ekleme (value veya provider)
int i = box.Add(42);
int s = box.Add("Merhaba");
int g = box.AddFunc(() => Guid.NewGuid().ToString("N"));

// Tip-güvenli okuma
int v = box.Get<int>(i);
if (box.TryGet<string>(s, out var text))
{
    // text = "Merhaba"
}

// Tip belirtmeden okuma (DynamicInvoke yok)
object any = box.GetVal(g);
string str = box.GetString(g);

// Güncelleme
box.Set(i, 100);
box.SetFunc(s, () => "Daha sonra!");

// Silme / temizleme
box.RemoveAt(g);
box.Clear();
Performans önerisi: Mümkünse Get<T> kullan; gerçekten gerekiyorsa GetVal/GetString’e düş.

DD Modeli: DDContainer → DDTable → DDRow

MagicBox; hücre konteynerinin üstüne, hafif bir tablo/satır modeli ekler. Böylece birden fazla tabloyu tek bir konteynerde tutabilir, kolon davranışını yönetebilirsiniz.

  • DDContainer: Birden fazla tabloyu taşır.
  • DDTable: Kolon seti + satır koleksiyonu.
  • DDRow: Satır içindeki hücreler (IDBox/handle yaklaşımı).
// Şablon örnek (kolon davranışı repo kurgusuna göre değişebilir)
var db = new DDContainer();
var table = db.AddTable("People", ColumnBehaviour.StrongType);

// Kolonlar tanımlanır, ardından satırlar eklenir.
// (Detayları repo içindeki isimlere göre netleştirebiliriz.)

API Özeti

En sık kullanılan fonksiyonların hızlı listesi:

// Ekleme
Add<T>(T value)
AddFunc<T>(Func<T> provider)
AddDelegate(Delegate d)

// Okuma
Get<T>(int index)
TryGet<T>(int index, out T value)
GetVal(int index)
TryGetVal(int index, out object value)
GetString(int index)

// Güncelleme
Set<T>(int index, T value)
SetFunc<T>(int index, Func<T> provider)
SetDelegate(int index, Delegate d)

// Silme / Temizleme
RemoveAt(int index)
Remove(IDBox cell)
Clear()
Sınırlama: Hücreler sadece Func<T> (0 parametre) mantığıyla tasarlanmıştır.
