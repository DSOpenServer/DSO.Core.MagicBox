MagicBox

MagicBox, değerleri zero-arg Func<T> hücreleri olarak tutan, DynamicInvoke kullanmadan tip-güvenli okuma sağlayan performans odaklı bir mini konteynerdir. Üzerine inşa edilmiş hafif DDContainer → DDTable → DDRow modeli içerir.

GitHub Repo Hızlı Başlangıç

Genel Bakış

Amaç: DynamicInvoke olmadan tip güvenliği + sıcak yolda düşük maliyet.

  • Get<T> / TryGet<T>
  • GetVal / GetString hızlı yol
  • Invoker cache ile performans
  • DDContainer tablo soyutlaması

Hızlı Başlangıç

var box = new DMagicBox();

int i = box.Add(42);
int s = box.Add("Merhaba");

int value = box.Get<int>(i);
string text = box.Get<string>(s);

object any = box.GetVal(i);
string str = box.GetString(s);

DD Modeli

DDContainer birden fazla tablo taşır.

var db = new DDContainer();
var table = db.AddTable("People", ColumnBehaviour.StrongType);
