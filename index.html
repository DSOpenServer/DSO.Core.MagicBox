MagicBox

MagicBox, değerleri sıfır parametreli Func<T> delege hücreleri olarak tutan, DynamicInvoke kullanmadan tip-güvenli okuma yapan, performans odaklı bir mini konteynerdir. Üzerine inşa edilmiş hafif bir satır/tablolaştırma katmanı da içerir. :contentReference[oaicite:4]{index=4}

Ne sağlar?

Tip-güvenli okuma, lazy değer üretimi, object/string hot-path invoker cache, basit tablo/satır modelleme (StrongType/Free).

Ne değildir?

Tam bir ORM veya veritabanı katmanı değildir. Minimal, taşınabilir bir çekirdek bileşen olarak konumlanır. :contentReference[oaicite:6]{index=6}

✨ Neler Sunuyor?

MagicBox’ın ana hedefi: DynamicInvoke olmadan tip güvenliği + sıcak yolda düşük maliyet. :contentReference[oaicite:7]{index=7}

  • Tür-güvenli çekim: Get<T>() / TryGet<T>(out T) ile casting veya DynamicInvoke olmadan doğru tipte okuyun. :contentReference[oaicite:8]{index=8}
  • Hızlı object/string erişimi: TryGetVal(out object) ve GetString() hot-path’te tip başına derlenen invoker cache kullanır. :contentReference[oaicite:9]{index=9}
  • Laziness: Hücreler Func<T>; gerçek değer ihtiyaç anında üretilir. :contentReference[oaicite:10]{index=10}
  • Tablo soyutlaması: DDContainer → DDTable → DDRow → IDBox zinciri; StrongType veya Free kolon davranışı. :contentReference[oaicite:11]{index=11}
  • Minimal bağımlılık: Sade .NET API’leri üzerinde; herhangi bir framework’e kilitli değil. :contentReference[oaicite:12]{index=12}

Hızlı Başlangıç

Değer saklama & okuma örneği (README’deki akış): :contentReference[oaicite:14]{index=14}

using DSO.Core.MagicBox;

var box = new DMagicBox();

// Hücre ekleme (capture eden closure ile):
int i = box.Add(42);                // Func<int>
int s = box.Add("Merhaba");         // Func<string>
int n = box.Add(DateTime.UtcNow);   // Func<DateTime>

// Sağlayıcı ile ekleme (lazy):
int r = box.AddFunc(() => Guid.NewGuid().ToString("N")); // Func<string>

// Tür-güvenli çekim
int fortyTwo = box.Get<int>(i);
string hello = box.Get<string>(s);
DateTime ts = box.Get<DateTime>(n);

// Tip uyumsuzluğu: TryGet ile güvenli kontrol
if (!box.TryGet<double>(i, out var dbl))
{
    // listedeki delegenin tipi double değil
}

// Tip belirtmeden object/string alma (DynamicInvoke YOK)
object any = box.GetVal(r);
string text = box.GetString(r);

// Hücre güncelleme
box.Set(i, 100);
box.SetFunc(s, () => "Daha sonra!");

// Silme & Temizleme
box.RemoveAt(n);
box.Clear();
IDBox/Handle ile çalışma: DMagicBox enumerator’ı IDBox döndürür; hücre üzerinde doğrudan Get<T>(), GetVal(), GetString() çağrıları yapılabilir. :contentReference[oaicite:15]{index=15} 
foreach (IDBox cell in box)
{
    // Not: T hücre tipiyle uyumlu olmalı
    // string sVal = cell.Get<string>();
    object oVal = cell.GetVal();
    string asText = cell.GetString();
}

Tablo Modeli: DDContainer → DDTable → DDRow → IDBox

DDContainer birden fazla tabloyu taşır; her DDTable kolon koleksiyonu ve satır koleksiyonuna sahiptir. :contentReference[oaicite:16]{index=16}

Kolon Davranışı
  • StrongType: Kolon tipleri şema gibi korunur; uyumsuz tip set edilirse açıklayıcı hata. :contentReference[oaicite:17]{index=17}
  • Free: Kolonlar esnektir; indeks sırasına göre doldurabilirsin. :contentReference[oaicite:18]{index=18}
Basit StrongType Örneği
using DSO.Core.MagicBox;

var db = new DDContainer();
var table = db.AddTable("People", ColumnBehaviour.StrongType);

// Şema: Kolonları önceden tanımlayın
var cols = table.ColumnNames;
cols.AddColumn("Id", typeof(int));
cols.AddColumn("Name", typeof(string));
cols.AddColumn("Birth", typeof(DateTime));

// (Devamında) row ekleyip kolonları doldurabilirsiniz

InvokerCache / Performans
  • DynamicInvoke yok: Tip özelinde derlenmiş invoker’lar kullanılır (InvokerCache). :contentReference[oaicite:21]{index=21}
  • String hot-path: Func<T> → string yolu Expression ile derlenip cache’lenir. :contentReference[oaicite:22]{index=22}
  • Boxing notu: Value-type’larda GetVal/GetString yolunda kaçınılmaz boxing olabilir. :contentReference[oaicite:23]{index=23}
Önemli: Sadece Func<T> (0 parametre) hücreleri desteklenir. :contentReference[oaicite:24]{index=24}

Sınırlamalar & Davranışlar

  • Yalnızca zero-arg: Hücreler sadece Func<T> (0 parametre) olarak desteklenir. :contentReference[oaicite:25]{index=25}
  • StrongType tip uyumsuzluğu: Beklenen tip dışında set etmeye çalışırsanız açıklayıcı hata. :contentReference[oaicite:26]{index=26}
  • İndeks sınırları: Negatif veya aralık dışı indeksler güvenle reddedilir. :contentReference[oaicite:27]{index=27}
  • Thread-safety: Default sınıflar thread-safe değildir. :contentReference[oaicite:28]{index=28}
